🔹 A. Context Risk (STATIC SNAPSHOT)

Represents current conditions

Inputs:
Feature	Description	Range
port_congestion	0–1	traffic at port
weather_severity	0–1	storm / delay risk
carrier_reliability	0–1	inverse risk
customs_delay	0–1	clearance issues
geopolitical_risk	0–1	region instability
Formula:
Context_Risk =
  0.3 * port_congestion +
  0.25 * weather_severity +
  0.2 * (1 - carrier_reliability) +
  0.15 * customs_delay +
  0.1 * geopolitical_risk
🔹 B. Temporal Risk (TREND / CHANGE)

Simulates LSTM behavior

Inputs:
delta_congestion (change over time)
delay_trend
route instability
Formula:
Temporal_Risk =
  0.5 * congestion_trend +
  0.3 * delay_trend +
  0.2 * route_variability

👉 If no history → fallback = Context_Risk * 0.5

🔹 C. Anomaly Risk (OUTLIER DETECTION)

Simulates Isolation Forest

Logic:
If sudden spike in any feature:
    anomaly_score = high
Else:
    anomaly_score = low
Example:
congestion jumps from 0.3 → 0.9 → HIGH anomaly
vessel stops unexpectedly → HIGH anomaly
Formula:
Anomaly_Risk =
  max(
    abs(current - avg_baseline),
    spike_indicator
  )

def compute_context_risk(data):
    return (
        0.3 * data["port_congestion"] +
        0.25 * data["weather"] +
        0.2 * (1 - data["carrier_reliability"]) +
        0.15 * data["customs_delay"] +
        0.1 * data["geo_risk"]
    )


def compute_temporal_risk(data):
    if "history" not in data:
        return compute_context_risk(data) * 0.5

    return (
        0.5 * data["trend_congestion"] +
        0.3 * data["trend_delay"] +
        0.2 * data["route_variability"]
    )


def compute_anomaly_risk(data):
    baseline = data.get("baseline", 0.5)
    current = data["port_congestion"]

    spike = abs(current - baseline)

    if spike > 0.5:
        return min(spike, 1.0)

    return spike * 0.5


def compute_dri_full(data):
    context = compute_context_risk(data)
    temporal = compute_temporal_risk(data)
    anomaly = compute_anomaly_risk(data)

    dri = (0.5 * context + 0.3 * temporal + 0.2 * anomaly) * 100

    return int(min(max(dri, 0), 100))
